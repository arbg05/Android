============================
*****Smart Weather 1.0*****
============================

- Displays Ambient temperature of the phone,
- Displays randomly generated temperature values (0 to 30) in a list view (Mon-Fri),
- Performs celsius to fahrenheit conversion using jni code built using ndk-build,
- Each temperature value is assigned with a weather icon
		- 0 to 9 Celsius   => Winter
		- 10 to 19 Celcius => Cloudy
		- 20 to 30 Celsius => summer
